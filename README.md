# Site SSMA

Projeto web desenvolvido com foco em apresentar uma solução digital voltada para a área de **SSMA (Saúde, Segurança, Meio Ambiente)**. O objetivo do site é oferecer uma interface clara, organizada e profissional para apresentar conteúdos, serviços e informações de forma acessível e visualmente agradável.

## 📌 Objetivo do projeto

Este projeto foi criado para simular um site institucional/profissional, com foco em organização de conteúdo, identidade visual e experiência do usuário. A proposta é demonstrar habilidades em desenvolvimento web, estruturação de páginas e construção de interfaces com propósito real.

## 🚀 Funcionalidades

- Página inicial com apresentação principal
- Estrutura de navegação entre seções
- Área de cursos/serviços
- Layout visual planejado para transmitir profissionalismo
- Design voltado para uma boa experiência do usuário
- Organização de conteúdo com foco em clareza e acessibilidade
